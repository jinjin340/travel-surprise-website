# 给小羊的下一站惊喜 ✈️

这是一个旅游目的地随机选择网站，为小羊同学提供惊喜的旅行计划。

## 功能特点

- 🌠 星空背景的开盒界面
- 🎁 随机抽取旅游目的地
- 📱 响应式设计，支持移动端
- 🗺️ 高德地图定位
- 📝 详细的5天4夜行程安排
- 🎨 小红书风格的详情页

## 如何使用

1. 打开网站
2. 点击「点击开盒 🎁」按钮
3. 等待随机抽取结果
4. 查看详细的旅行攻略

## 技术栈

- HTML5 + CSS3 + JavaScript
- 高德地图 API
- 纯 CSS 动画效果
